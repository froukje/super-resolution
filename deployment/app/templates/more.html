Some Infos


This is a personal project.


The picture of the lizzard was downloaded from Chameleon photo created by kuritafsheen77 - www.freepik.com.


The increased resolution of the image is achieved using a super resolution neural network. The network architecture is based on the generator network of the paper Photo-Realistic Single Image Super-Resolution Using a Generative Adversarial Network. The code from their github has been adapted and transformed to pytorch-lightning. The complete code of the model, the preprocessing and this web app can be found on my github.
